# Minesweeper For DSA
## Introduction
This project is a 2D Platformer game developed using Unity and Visual Scripting (formerly Bolt)
## Documentation
### How to play:
Move Left / Right:
Use the "A" or "D" movement keys to control the character.
Collect Trophy:
Touch Trophy to complete level.

Jump:
Press the jump key to leap over gaps and obstacles.
### Features:
- Completed: 
    + Player movement using Visual Scripting
    + Jumping and gravity handling
    + Idle / Run / Jump animations
    + Camera following the player
    + Coin collection system
    + Collision detection with ground and walls
    + Basic physics-based character control
- Future plans: 
    + Register/ Login Feature.
    + Show top 10 scores for selected difficult level.
    + Add enemies, checkpoints and score.
